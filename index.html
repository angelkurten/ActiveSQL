Desarolladores

Quieres razones para utilizar ActiveSQL?

Olvidate del SQL

Si utilizas ActiveSQL te olvidaras de escribir esas que son algunas veces tediosas lineas de codigo en SQL e incluso puedes tender a escribirlas mal, nosotros lo hacemos por ti :D

Facil integracion

Integrar ActiveSQL en tu proyecto es muy facil, solo tienes que requerir y/o incluir un archivo en tu proyecto y estas listo :D

Olvidate del trabajo duro

ActiveSQL hace todo el trabajo duro por ti, se encarga de crear, ejecutar y cerrar la conexionc con la base de datos de forma automatica :D

Seguridad al instante

Al utilizar esta libreria automaticamente todos las parametros de tus consultas seran examinados en busca de Inyecciones SQL, en caso de encontrar algo ActiveSQL se encargara de purificar dicha consulta :D

Servicios RESTful(futuro)

Se tiene pensando incluir en un futuro proximo, incluir la posibilidad de crear servicios RESTful de una forma sencilla, de echo en la version actual ya se encuentra un pequeño webservices funcionando con la base de datos a modo de prueba, mas adelante ampliaremos esta caracteritica.

ActiveSQL es Libre

Si señor@s, puedes sentirte en la libertad de modificar y distribuir esta libreria siempre y cuando no te olvides de notificar quien es el autor original de la libreria :D

Ejemplos Basicos

Estos son ejemplos 4 ejemplos basicos de como usar ActiveSQL

NOTA: La variable $ac es el objeto que contiene todas los metodos de ActiveSQL

SELECT

//array con los campos a seleccionar
$array= array('nombre', 'email');
//envio los campos a seleccionar
$ac->select($array);
//construyo y ejecuto la consulta, me duelve un valor segun yo solo solicite
//json, xml, object y fetch son las opciones disponibles
$user=$ac->get('usuarios')-json();

UPDATE

//array con el filtro where
$array=array('id'=>'1');
//guardo los campos a filtrar
$ac->where($array);
//array con los datos a actualizar
$array=array('nombre'=>'Angel','apellido'=>'Kurten');
//guardo los campos a actualizar
$ac->set($array);
//construyo y ejecuto la consulta
$ac->edit('usuarios');

INSERT

//array con los datos a guardar
$array=array('0',"Angel", 'Kurten', 'angelkurten@hotmail.com');
//guardo los campos a guardar
$ac->values($array);
//construyo y ejecuto la consulta
$ac->save('usuarios');

DELETE

//array con el filtro where
$array=array('id'=>'1');
//ejecuto la consulta
$ac->delete('usuarios');

Incluye el archivo active.php y crea un objeto para que puedas empezar a trabajar.

NOTA 1: En este caso utilizaremos el metodo require_once de PHP estas en la libertad de utilizar otro

creamos el objeto $ac con todos los metodos de ActiveSQL
$ac=new active;

Metodos del objeto y sus parametros

Aqui esta toda la documentacion posible de los metodos disponibles en ActiveSQl con los parametros que acepta.

Metodo Tipo parametro(# de parametros) Descripcion
get('nombre tabla'); string(1) Metodo para realizar consultar registros a la base de datos
save('nombre tabla') string(1) Metodo para guardar registros en la base de datos
edit('nombre tabla') string(1) Metodo para editar registros en la base de datos
delete('nombre tabla') string Metodo para eliminar registros en la base de datos

Metodos complementarios, solo que no funcionan si no realizamos una de las acciones anteriores

Metodo Tipo parametro(# de parametros) Descripcion
set() array(1) Metodo que complementa la funcion edit, al cual se envia un array que contiene los nombres y el valor de los campos a actualizar
select() array(1) Metodo que nos permite seleccionar campos en especifico de una tabla
values() array(1) Metodo que complementa la funcion insert, que nos definir valores para campos en una tabla
where() array(1) Metodo que nos permite aplicar filtros a nuestas consultas

FAQ Archivos

  • index.php
  • test.php
  • database.sql

  • Carpeta Core

    • active.php
    • db.php
    • lib.php
active.php
Este archivo contiene todas una clase con el mismo nombre, y dentro de ella tiene todos los metodos publicos que nos permite realizar acciones detro de nuestra base de datos...
db.php
Este archivo es el corazon de la aplicacion, es el encargado de definir los metodos que utiliza active.php, y de realizar la parte pesada del trabajo, como conectarnos con la base de datos, ejectutar la consulta, devolver valores consultados, y cerrar la conexion.
lib.php
Este archivo es un extra de la liberia, el cual contiene varias funciones para medir el tiempo en el que se ejecutan las consultas, para realizar inclusiones de archivos de una forma mas comoda y sencila, ademas de una funcion para mostrar los errores de PHP de una forma diferente.
index.php
Archivo en el cual hay varios ejemplos de como usar ActiveSQL, en la primera implementacion te muestra el tiempo que tardo en ejecutarse esto lo puedes comparar con el resultado de test.php el cual realiza la misma operacion pero de la forma tradicional
test.php
Realiza la primera operacion de seleccion de datos realizada en el index.php y su tiempo de ejecucion para lo compares con el del index.php
database.sql
Este archivo contiene una tabla de usuarios con unos datos de prueba los cuales debes importar en una base de datos llamada active
